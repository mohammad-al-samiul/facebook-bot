# বট-এজেন্ট প্রজেক্ট — সম্পূর্ণ বাংলা নির্দেশিকা

এই ডকুমেন্টে রয়েছে এই প্রজেক্টে যা যা করা হয়েছে তার পূর্ণ বিবরণ — কোন প্রযুক্তি কেন বেছে নেওয়া হয়েছে, কোন ফাইল কী কাজ করে, এবং কোন স্ক্রিপ্ট কীভাবে চালাতে হবে।

---

## ১. প্রজেক্টের উদ্দেশ্য

এটি একটি **পাইথন-ভিত্তিক ব্রাউজার অটোমেশন ফ্রেমওয়ার্ক** যা ফেসবুক অ্যাকাউন্টে মানুষের মতো আচরণ (স্ক্রল, লাইক, কমেন্ট) করতে পারে। প্রজেক্টের প্রধান দিকগুলো হলো —

- প্রতিটি অ্যাকাউন্টের জন্য আলাদা স্থায়ী ক্রোমিয়াম প্রোফাইল (কুকি ও সেশন ডিস্কে সংরক্ষিত)
- মোবাইল ডিভাইস এমুলেশন (৩৬০×৮০০ ভিউপোর্ট, মোবাইল ইউজার-এজেন্ট, টাচ ও স্কেল ফ্যাক্টর)
- ফিঙ্গারপ্রিন্ট মাস্কিং — `playwright-stealth` সহ ক্যানভাস/ওয়েবজিএল/ওয়েবআরটিসি স্পুফিং
- মানুষের মতো ইনপুট — বেজিয়ার কার্ভে মাউস মুভমেন্ট, অক্ষর-অক্ষর টাইপিং, র‍্যান্ডম বিরতি
- জেমিনি এআই দিয়ে স্বয়ংক্রিয় কমেন্ট জেনারেট (বাংলা/ইংরেজি ভাষা অনুসারে)
- একই পোস্টে দ্বিতীয়বার ইন্টারঅ্যাকশন না করার ব্যবস্থা (সেশন-ভিত্তিক ডিডুপ্লিকেশন)
- প্রতি ৫টি ইন্টারঅ্যাকশনের পর ৫–১০ মিনিটের কুলডাউন
- মঙ্গোডিবি দিয়ে একাধিক বটের ফ্লিট চালানোর সুবিধা (ঐচ্ছিক)

> **গুরুত্বপূর্ণ সতর্কতা:** ফেসবুক বা যেকোনো সোশ্যাল প্ল্যাটফর্মের টার্মস অফ সার্ভিস মেনে চলা আপনার দায়িত্ব। এই টুল শুধু একটি টেকনিক্যাল অটোমেশন কাঠামো। ভুল ব্যবহার করলে আপনার অ্যাকাউন্ট স্থায়ীভাবে ব্যান হতে পারে।

## ২. ব্যবহৃত প্রযুক্তি ও কেন

| প্রযুক্তি                 | ভূমিকা ও কেন বেছে নেওয়া হয়েছে                                                                                                         |
| ------------------------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| **পাইথন ৩.১০ বা পরবর্তী** | অ্যাসিঙ্ক প্রোগ্রামিং, টাইপ হিন্ট, আধুনিক ভাষাগত সুবিধা                                                                                 |

| **certifi**               | বিকল্প হিসেবে CA bundle (truststore না থাকলে ব্যবহৃত)                                                                                   |
| **tzdata**                | `zoneinfo` দিয়ে সময় অঞ্চল হ্যান্ডলিং (উইন্ডোজে বিশেষভাবে দরকার)                                                                       |
| **Docker (ঐচ্ছিক)**       | লোকাল মঙ্গোডিবি চালাতে — `docker-compose.yml` দেওয়া আছে                                                                                |
| **Google Gemini API**     | পোস্টের ভাষা বুঝে স্বল্প-শব্দের প্রাকৃতিক কমেন্ট তৈরি করে                                                                               |

#### `actions.py` ভেতরে কী কী আছে

| ফাংশন                             | কাজ                                                                                                                                        |
| --------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------ |
| `human_click(page, element)`      | এলিমেন্টে কার্ভড পথে মাউস নিয়ে গিয়ে স্বাভাবিকভাবে ক্লিক                                                                                  |
| `human_scroll(page, ...)`         | দ্রুত মাউস হুইল স্ক্রল বার্স্ট, ভ্যারিয়েবল গতিতে                                                                                          |
| `human_like_scroll(page, ...)`    | **৩০০–৭০০ পিক্সেল র‍্যান্ডম স্ক্রল, ২–৫ সেকেন্ড বিরতি** — পোস্ট পড়ার মতো ধীর গতি                                                          |
| `human_type(page, element, text)` | **প্রতিটি অক্ষরে ১০০–৩০০ মিলিসেকেন্ড বিরতি দিয়ে টাইপ**                                                                                    |
| `random_delay(min, max)`          | দুই সেকেন্ড সংখ্যার মধ্যে ঘুম                                                                                                              |
| `react_to_post(...)`              | বহুভাষিক Like বাটন খুঁজে রিঅ্যাকশন দেওয়া                                                                                                  |
| `comment_on_post(...)`            | কমেন্ট বাটনে ক্লিক → টেক্সটবক্স খোঁজা → মানুষের গতিতে টাইপ → **৪-স্তরের সাবমিট লজিক** (বাটন ক্লিক → Ctrl+Enter → Enter → পুনরায় বাটন চেক) |
| `GENERIC_COMMENTS`                | এআই ফেইল হলে ব্যবহার্য কমেন্টের তালিকা (ইমোজি ও সংক্ষিপ্ত বাক্য)                                                                           |

#### `ai_comment.py` কী করে

- `.env` থেকে `GEMINI_API_KEY` পড়ে
- পোস্টের টেক্সটে বাংলা অক্ষর আছে কিনা দেখে (`U+0980` থেকে `U+09FF` রেঞ্জ)
- বাংলা থাকলে বাংলায়, না হলে ইংরেজিতে একটা **ছোট, প্রাকৃতিক, মানুষের মতো** কমেন্ট চাইতে জেমিনিকে অনুরোধ পাঠায়
- TLS সমস্যা এড়াতে উইন্ডোজে `truststore`-এর নেটিভ সার্ট স্টোর ব্যবহার করে
- `thinkingBudget: 0` সেট করে, কারণ Gemini 3 Flash প্রিভিউ মডেল ডিফল্ট-এ "thinking" টোকেন ব্যবহার করে যা আউটপুট বাজেট খেয়ে নেয়
- নেটওয়ার্ক/এপিআই যেকোনো সমস্যা হলে `"Good post!"` রিটার্ন করে — বট বন্ধ হয় না

#### `post_engagement.py` কী করে

১. বর্তমান পেজে পোস্ট কন্টেইনার খোঁজে — প্রথমে `[role="article"]`, না পেলে দৃশ্যমান Like বাটনগুলোকে অ্যাঙ্কর ধরে DOM-এ উপরের দিকে হাঁটে যতক্ষণ না সত্যিকারের পোস্ট প্যারেন্ট পায়
২. প্রতিটি পোস্টের দৃশ্যমান টেক্সট বের করে SHA-1 হ্যাশ বানায় — সেটাই হলো **ফিঙ্গারপ্রিন্ট**
৩. সেশনে দেখা ফিঙ্গারপ্রিন্ট সেটে চেক করে — আগে দেখা থাকলে স্কিপ
৪. ২–৫ সেকেন্ড "পড়ার বিরতি"
৫. কনফিগার করা প্রোবাবিলিটিতে (ডিফল্ট ৭০%) Like দেয়
৬. আলাদা প্রোবাবিলিটিতে (ডিফল্ট ৪০%) `get_ai_comment()` কল করে কমেন্ট পোস্ট করে
৭. পোস্টের মাঝে অতিরিক্ত মানব-সদৃশ স্ক্রল

## ৬. লগ ফাইল ও সমস্যা সমাধান

### লগ কোথায়

- কনসোলে চলমান সব লগ
- ফ্লিট চালালে `logs\fleet_main.log` ও `logs\<account_id>.log`
- AI বটে `Start-Process` দিয়ে চালালে `logs\ai_bot_run.log`

সব লগ **ইংরেজিতে** — উইন্ডোজে ফাইল রিড সহজ হয়।

### "Feed did not render within 45s"

ফেসবুক এই অ্যাকাউন্টে টেম্পোরারি **থ্রটল** লাগিয়েছে — পরপর অনেক লাইকের পর ফিড সাময়িকভাবে স্থগিত। সমাধান —

- কয়েক ঘণ্টা অপেক্ষা করুন (কখনো ১২–২৪ ঘণ্টা)
- পরবর্তীতে কম `--like-chance` ও বড় `--cooldown-after` ব্যবহার করুন

### "Comment skipped (no editor located)"

কমেন্ট বাটনে ক্লিকের পর এডিটর-টেক্সটবক্স পাওয়া যায়নি। মোবাইল ফেসবুকে কখনো কখনো হয়। বট এটাকে বিপজ্জনক বিবেচনা করে স্কিপ করে — পরের পোস্টে যায়।

### "Gemini API HTTP 429"

জেমিনির ফ্রি কোটা শেষ। `"Good post!"` ফলব্যাক ব্যবহৃত হবে — বট থামবে না। কয়েক ঘণ্টা পর কোটা রিসেট হয়।

### "MongoDB ServerSelectionTimeoutError"

- Atlas Primary নোড ফ্ল্যাপ করছে
- অথবা আপনার আইএসপি পোর্ট ২৭০১৭ ব্লক করছে (বাংলাদেশের অনেক আইএসপি করে)
- পরামর্শ: ভিপিএন ব্যবহার করুন (Cloudflare WARP, Proton ইত্যাদি)
- বিকল্প হিসেবে `run_ai_bot.py` ব্যবহার করুন — মঙ্গো লাগে না

### "SSL: CERTIFICATE_VERIFY_FAILED"

`truststore` ইনস্টল না থাকলে ঘটতে পারে —

```powershell
pip install truststore
```

পুনরায় চালান। আমাদের কোড স্বয়ংক্রিয়ভাবে উইন্ডোজ নেটিভ সার্ট স্টোর ব্যবহার করে।

### "fleet_worker.py" সরাসরি চালালে এরর

পাওয়ারশেলে পাইথন ফাইল সরাসরি কমান্ড হিসেবে চলে না। সব সময় —

```powershell
python <স্ক্রিপ্ট_পাথ>
```

লিখতে হবে।

২. **প্রতি ঘণ্টায় খুব বেশি অ্যাকশন এড়িয়ে চলুন** — বাস্তব মানুষ একটানা ৩০ লাইক দেয় না
৩. **একই আইপি থেকে একসাথে অনেক অ্যাকাউন্ট ব্যবহার করবেন না** — প্রতিটি অ্যাকাউন্টে আলাদা প্রক্সি (`bots` ডকুমেন্টের `proxy` ফিল্ড) দিন
৪. **মোবাইল কুকি ও মোবাইল প্রোফাইল একসাথে ব্যবহার করুন** — ডিভাইস মিসম্যাচ ক্যাপচার মূল কারণ
৫. **লগ পর্যবেক্ষণ করুন** — "Account restricted" বার্তা দেখলে সেই অ্যাকাউন্ট তৎক্ষণাৎ বন্ধ করুন

| ফিচার                                      | কখন/কেন                                                                                                               |
| ------------------------------------------ | --------------------------------------------------------------------------------------------------------------------- |
| `cookies.txt`-ভিত্তিক single-account রানার | মঙ্গোডিবি ছাড়াই দ্রুত পরীক্ষার জন্য                                                                                  |
| মোবাইল UA + ৩৬০×৮০০ ভিউপোর্ট ডিফল্ট        | কুকি মোবাইল সেশন থেকে আসা, ডিভাইস মিল রাখতে                                                                           |
| `stealthy_facebook_login`                  | মানুষের গতিতে টাইপ + কার্ভড মাউস, যাতে ক্যাপচা না আসে                                                                 |
| বহুভাষিক বাটন ডিটেকশন                      | অনেক অ্যাকাউন্ট ইন্দোনেশিয়ান লোকেলে, তাই Like/Comment এর aria-label বাংলা/ইংরেজি/ইন্দোনেশিয়ানসহ একাধিক ভাষায় ম্যাচ |

| `truststore` ইন্টিগ্রেশন                   | উইন্ডোজে TLS সার্ট ভেরিফাই ব্যর্থ হলে নেটিভ স্টোর ব্যবহার                                                             |
